What is this?

This project provides you with a cli tool that you can use to generate source code files in your project, You will have a file named atomicg.yml in the root of your project, in this file you can put the entities you want to generate for your project, you also can specify what generators must be use. You can run the tool from the command line to generate the source code files that match your configuration. You can repeat this process indefinitely to always have a functional well formatted and well written model of data for your project.

How it Works?

The tool works with our remote server to create the source code files for your project. First, the cli client tool collects the information from your local file system, namely it will read your model and send it to the server. Then the server will process your model and generated the relevant source code files. Finally the server will send the generated source code files to the cli tool so it can put the files in your projects in their respective locations.

MySQL DDL script

Generates the necessary DDL script for MySQL to create and upgrade the database of your project.

Sequelize models

Generates the sequelize models for your entities so they can be use in your business logic.

ExpressJS CRUDs

Creates a CRUD router for expressjs for each entity so it can be use as part of the api of your backend.
